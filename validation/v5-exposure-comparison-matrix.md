# V5 Exposure Comparison Matrix

## Purpose

This file is the compact comparison matrix for the V5 high-heat coupon exposure stage.

It exists to keep the stage centered on the real question:

> Does the best surviving passive stack remain more credible than simpler passive comparisons under severe high-heat exposure?

---

## Candidate Set

| Article ID | Layer State | Primary Comparison Purpose |
|---|---|---|
| V5-P1 | Best surviving passive stack family | Primary severe passive truth article |
| V5-B1 | Less-decoupled or simplified passive comparison | Determine whether the full passive stack earns its complexity |
| V5-B2 | Earlier-stage baseline passive comparison | Preserve continuity with prior stage logic |
| V5-R1 | Boundary-penalized passive coupon | Expose boundary vulnerability under severe heat |
| V5-R2 | Alternate top-ranked passive stack | Prevent false certainty if two passive routes remain competitive |

---

## Ranking Categories

Use the following first-pass ranking categories when V5 is eventually executed.

### Rank A
Strong continuation candidate

### Rank B
Credible continuation candidate with caveats

### Rank C
Useful comparison result but not preferred for continuation

### Rank D
Cut or deprioritize

---

## Minimum Evaluation Questions

Each article family should be judged against these questions.

1. Did it remain grossly intact under severe exposure?
2. Did it stay more coherent than simpler passive comparisons?
3. Did the thermal trend remain interpretable?
4. Did it reveal a believable passive-band boundary?
5. Is it worth carrying into V6?

---

## Planned Advancement Logic

### Advance by default
- Rank A
- Rank B

### Advance only if needed for comparison continuity
- Rank C

### Cut unless a very specific reason exists
- Rank D

---

## Stage Discipline Rule

No passive stack family may advance merely because the repo wants the passive-first story to survive.

Advancement must be tied to comparative severe-exposure performance.
